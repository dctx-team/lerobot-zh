# SO-101

在以下步骤中，我们将说明如何组装我们的旗舰机器人 SO-101。

## 采购零件

请遵循此 [README](https://github.com/TheRobotStudio/SO-ARM100)。它包含物料清单、零件采购链接以及 3D 打印零件的说明。
如果这是您第一次打印或您没有 3D 打印机，文档中也提供了建议。

## 安装 LeRobot 🤗

要安装 LeRobot，请遵循我们的 [安装指南](./installation)

除了这些说明之外，您还需要安装 Feetech SDK：

```bash
pip install -e ".[feetech]"
```

## 分步组装说明

follower 机械臂使用 6 个带有 1/345 齿轮比的 STS3215 电机。然而，leader 使用三个不同齿轮比的电机，以确保它既能支撑自身重量，又能在不需要太多力的情况下移动。下表显示了每个关节所需的电机。

| Leader 机械臂轴  | 电机 | 齿轮比   |
| --------------- | :--: | :------: |
| Base / Shoulder Pan |   1   |  1 / 191   |
| Shoulder Lift       |   2   |  1 / 345   |
| Elbow Flex          |   3   |  1 / 191   |
| Wrist Flex          |   4   |  1 / 147   |
| Wrist Roll          |   5   |  1 / 147   |
| Gripper             |   6   |  1 / 147   |

### 清洁零件

从 3D 打印零件上移除所有支撑材料。最简单的方法是使用小螺丝刀到支撑材料下方。

建议在继续组装之前，在将电机放置后在电机中安装一根 3 针电缆。

### 关节 1

- 将第一个电机放入底座。
- 用 4 颗 M2x6mm 螺丝（最小的螺丝）固定电机。顶部两颗，底部两颗。
- 滑入第一个电机固定器并使用两颗 M2x6mm 螺丝固定（每侧一颗）。
- 安装两个电机角，用 M3x6mm 螺丝固定顶部的角。
- 连接肩部零件。
- 用顶部 4 颗 M3x6mm 螺丝和底部 4 颗 M3x6mm 螺丝拧紧肩部零件
- 添加肩部电机固定器。

## 配置电机

### 1. 查找与每个机械臂关联的 USB 端口

要查找每个总线伺服适配器的端口，请通过 USB 和电源将 MotorBus 连接到您的计算机。运行以下脚本并在提示时断开 MotorBus：

```bash
lerobot-find-port
```

<hfoptions id="example">
<hfoption id="Mac">

示例输出：

```
Finding all available ports for the MotorBus.
['/dev/tty.usbmodem575E0032081', '/dev/tty.usbmodem575E0031751']
Remove the USB cable from your MotorsBus and press Enter when done.

[...断开相应的 leader 或 follower 机械臂并按 Enter...]

The port of this MotorsBus is /dev/tty.usbmodem575E0032081
Reconnect the USB cable.
```

找到的端口是：`/dev/tty.usbmodem575E0032081`，对应您的 leader 或 follower 机械臂。

</hfoption>
<hfoption id="Linux">

在 Linux 上，您可能需要通过运行以下命令授予对 USB 端口的访问权限：

```bash
sudo chmod 666 /dev/ttyACM0
sudo chmod 666 /dev/ttyACM1
```

示例输出：

```
Finding all available ports for the MotorBus.
['/dev/ttyACM0', '/dev/ttyACM1']
Remove the usb cable from your MotorsBus and press Enter when done.

[...断开相应的 leader 或 follower 机械臂并按 Enter...]

The port of this MotorsBus is /dev/ttyACM1
Reconnect the USB cable.
```

找到的端口是：`/dev/ttyACM1`，对应您的 leader 或 follower 机械臂。

</hfoption>
</hfoptions>

### 2. 设置电机 ID 和波特率

总线上的每个电机都由唯一的 ID 标识。全新电机通常带有默认 ID `1`。为了使电机与控制器之间的通信正常工作，我们首先需要为每个电机设置唯一的、不同的 ID。此外，总线上数据传输的速度由波特率决定。为了彼此通信，控制器和所有电机都需要配置为相同的波特率。

为此，我们首先需要使用控制器单独连接到每个电机以设置这些参数。由于我们将这些参数写入电机内部存储器（EEPROM）的非易失性部分，我们只需执行一次此操作。

如果您正在重新利用另一个机器人的电机，您可能还需要执行此步骤，因为 ID 和波特率可能不匹配。

下面的视频显示了设置电机 ID 的步骤顺序。

#### Follower

将 USB 线从计算机和电源连接到 follower 机械臂的控制器板。然后，使用您从上一步获得的端口运行以下命令或运行 API 示例。您还需要使用 `id` 参数为您的 leader 机械臂命名。

<hfoptions id="setup_motors">
<hfoption id="Command">

```bash
lerobot-setup-motors \
    --robot.type=so101_follower \
    --robot.port=/dev/tty.usbmodem585A0076841  # <- 在此处粘贴上一步找到的端口
```

</hfoption>
<hfoption id="API example">

<!-- prettier-ignore-start -->
```python
from lerobot.robots.so101_follower import SO101Follower, SO101FollowerConfig

config = SO101FollowerConfig(
    port="/dev/tty.usbmodem585A0076841",
    id="my_awesome_follower_arm",
)
follower = SO101Follower(config)
follower.setup_motors()
```
<!-- prettier-ignore-end -->

</hfoption>
</hfoptions>

您应该看到以下说明

```bash
Connect the controller board to the 'gripper' motor only and press enter.
```

按照说明，插入夹持器的电机。确保它是连接到控制板的唯一电机，并且电机本身尚未菊花链连接到任何其他电机。当您按下 `[Enter]` 时，脚本将自动设置该电机的 ID 和波特率。

<details>
<summary>故障排除</summary>

如果此时出现错误，请检查您的电缆并确保它们正确插入：

<ul>
  <li>电源</li>
  <li>计算机与控制器板之间的 USB 线缆</li>
  <li>从控制器板到电机的 3 针电缆</li>
</ul>

如果您使用的是 Waveshare 控制器板，请确保两个跳线设置在 `B` 通道（USB）上。

</details>

然后您应该看到以下消息：

```bash
'gripper' motor id set to 6
```

接着是下一条说明：

```bash
Connect the controller board to the 'wrist_roll' motor only and press enter.
```

您可以从控制器板断开 3 针电缆，但可以将其另一端连接到夹持器电机上，因为它已经在正确的位置。现在，将另一条 3 针电缆插入腕部滚动电机并将其连接到控制器板。与上一个电机一样，请确保它是连接到控制板的唯一电机，并且电机本身未连接到任何其他电机。

按照说明对每个电机重复此操作。

> [!TIP]
> 在按 Enter 之前检查每一步的接线。例如，在您操作电路板时，电源电缆可能会断开。

完成后，脚本将简单地结束，此时电机已准备好使用。您现在可以将每个电机的 3 针电缆插入下一个电机，并将第一个电机（ID=1 的 'shoulder pan'）的电缆插入控制器板，控制器板现在可以连接到机械臂的底座。

## 校准

接下来，您需要校准您的机器人，以确保 leader 和 follower 机械臂在相同物理位置时具有相同的位置值。
校准过程非常重要，因为它允许在一个机器人上训练的神经网络在另一个机器人上工作。

#### Follower

运行以下命令或 API 示例来校准 follower 机械臂：

<hfoptions id="calibrate_follower">
<hfoption id="Command">

```bash
lerobot-calibrate \
    --robot.type=so101_follower \
    --robot.port=/dev/tty.usbmodem58760431551 \ # <- 您的机器人的端口
    --robot.id=my_awesome_follower_arm # <- 为机器人指定唯一名称
```

</hfoption>
<hfoption id="API example">

<!-- prettier-ignore-start -->
```python
from lerobot.robots.so101_follower import SO101FollowerConfig, SO101Follower

config = SO101FollowerConfig(
    port="/dev/tty.usbmodem585A0076891",
    id="my_awesome_follower_arm",
)

follower = SO101Follower(config)
follower.connect(calibrate=False)
follower.calibrate()
follower.disconnect()
```
<!-- prettier-ignore-end -->

</hfoption>
</hfoptions>
