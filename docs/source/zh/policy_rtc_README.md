# Real-Time Chunking (RTC)

本模块包含 **Real-Time Chunking (RTC)** 的 LeRobot 实现，这是一种用于基于流匹配的策略的推理时技术。

**注意**：RTC 本身不是一个策略，而是一种推理增强技术，适用于基于流匹配的策略，包括 [π₀](../pi0/)、[π₀.₅](../pi05/) 和 [SmolVLA](../smolvla/)。
